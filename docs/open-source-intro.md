# CoC7 车卡器开源介绍

## 我们在做什么

这是一个基于《克苏鲁的呼唤 第七版》规则的手机端车卡工具。它不是规则百科，也不是跑团平台，而是一个“快速成卡”的流程型应用。

目标是让玩家在几分钟内完成可用角色卡，同时在关键节点获得明确、可理解的规则提示。

## 设计方向

### 1) 电子游戏感

- 分章节推进：每一章都有明确目标与反馈。
- 关键状态可见：属性池、技能池、规则提醒实时展示。
- 低阻力操作：尽量用触屏友好的按钮、卡片、列表完成选择。

### 2) 易懂

- 新手可用：不要求先完整阅读规则书。
- 软校验：提醒风险，但不强制阻断流程。
- 文案直给：尽量避免晦涩术语和过长说明。

### 3) 手机优先

- Web 与 Capacitor 共用业务逻辑。
- 导出在移动端可直接保存与分享。
- 样式和交互优先适配窄屏。

## 开源价值

- 规则可审查：核心计算和校验逻辑公开透明。
- 便于扩展：职业/武器/经历包以结构化数据维护。
- 便于协作：前端流程、规则层、导出层职责清晰。

## 后续方向

- 继续优化移动端交互与性能，强化“电子游戏感”反馈。
- 提供更完整的导出校验与回归测试样例。
- 为后续多规则系统（例如 DND）保留并行扩展能力。
