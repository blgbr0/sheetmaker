# CoC7 调查员建卡工坊

这是一个纯前端离线可跑的《克苏鲁的呼唤 第七版》建卡工具。

## 当前版本能力

- 分阶段流程（序章 → 第一章 → 第二章 → 第三章 → 终章）
- 职业图鉴墙（按 Excel 原顺序展示）：职业名 + 职业点公式 + 关键技能数量
- 点击职业卡查看完整职业详情（职业技能文本、必选技能、可选组、自选项）
- 选择职业时支持“清空职业点 / 保留当前加点”提醒
- 技能页职业技能置顶高亮，并对规则不满足项实时警告
- 可选技能使用标签按钮交互（不是下拉、不是复选框）
- 武器侧栏拖拽到角色卡，支持数量累加与移除
- 导出：
  - PDF：精简版简表
  - Excel：优先复用模板并写入核心命名字段，同时附加导出数据表

## 运行

```powershell
.\serve.ps1
```

打开：`http://localhost:5173`

## 说明

- 职业与武器数据优先尝试读取项目中的 `沙文.弗朗斯-逃兵-小乔.xlsx`。
- 若模板读取失败，会自动回退到内置基础职业/武器数据。
- 技能规则采用软校验：会提醒，但不强行阻止你进入下一步。
